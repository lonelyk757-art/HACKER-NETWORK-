📚 Learning Center

Structured paths from beginner to expert

Master cybersecurity through hands-on learning and practical experience

Beginner Path

Start your cybersecurity journey with foundational knowledge

1

Networking Fundamentals

Learn the basics of networking: TCP/IP, DNS, HTTP/HTTPS, ports, and protocols. Understand how data travels across networks.

Topics: OSI Model, IP Addressing, Subnetting, Routing

2

Linux Essentials

Master Linux command line, file systems, permissions, and basic administration. Linux is essential for security professionals.

Topics: Shell Commands, File Management, User Permissions, Package Management

3

Security Fundamentals

Understand core security concepts: confidentiality, integrity, availability, authentication, and authorization.

Topics: CIA Triad, Encryption, Hashing, Digital Signatures

4

Web Basics

Learn how web applications work: HTML, CSS, JavaScript, HTTP requests, and client-server architecture.

Topics: Web Architecture, HTTP Methods, Sessions, Cookies

Intermediate Path

Build practical skills and hands-on experience

1

Network Penetration Testing

Learn reconnaissance, scanning, enumeration, and exploitation techniques for networks and services.

Tools: Nmap, Metasploit, Wireshark, Burp Suite

2

Web Application Security

Master web vulnerabilities: SQL Injection, XSS, CSRF, authentication bypasses, and API security.

Topics: OWASP Top 10, Burp Suite, Payload Crafting

3

System Hardening

Learn to secure operating systems, configure firewalls, manage access controls, and implement security policies.

Topics: Firewall Rules, SELinux, AppArmor, Security Baselines

4

Threat Monitoring & SIEM

Understand log analysis, SIEM platforms, and threat detection techniques for identifying attacks.

Tools: Splunk, ELK Stack, Suricata, Osquery

Advanced Path

Specialize and master advanced techniques

1

Exploit Development

Learn to develop custom exploits, understand vulnerability classes, and perform reverse engineering.

Topics: Buffer Overflows, Format Strings, ROP Chains, Shellcode

2

Incident Response & Forensics

Master incident investigation, evidence collection, timeline analysis, and forensic analysis techniques.

Topics: Memory Forensics, Disk Forensics, Log Analysis, Chain of Custody

3

Advanced Threat Hunting

Learn to proactively search for threats, analyze attack patterns, and develop detection strategies.

Topics: MITRE ATT&CK, Threat Intelligence, Behavioral Analysis

4

Security Architecture

Design secure systems, implement defense-in-depth strategies, and manage enterprise security infrastructure.

Topics: Zero Trust, Cloud Security, Identity Management, Risk Management

Study Tips for Success

1. Practice Hands-On Labs

Theory alone isn't enough. Set up virtual labs, practice exploits, and build defensive systems. The more you practice, the better you'll understand.

2. Build a Home Lab

Create your own testing environment with virtual machines. Experiment safely and learn by doing.

3. Document Your Learning

Keep notes, create checklists, and document your discoveries. This helps reinforce learning and builds a personal knowledge base.

4. Stay Updated

Follow security news, read vulnerability disclosures, and stay informed about emerging threats and techniques.

5. Join the Community

Connect with other learners, participate in discussions, and share knowledge. Learning together accelerates growth.
